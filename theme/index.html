DAEFROM

is a global
open-source blockchain platform.

Awesome Features

DAEFROM technology stack can benefit in any use cases that demand security and decentralization as
open finance, personal identification, gaming, sensitive data and many others.
DAEFROM network runs a proof-of-work (PoW) consensus protocol.

Contracts & Tokens

DAEFROM network is ready for creating blockchain programs and tokenize anything you like. We are compatible with a wide selection of Ethereum contracts. Every mined DAE coin will have a pair DAEFROM token in a certain ratio based on our blockchain.

Social network future

DAEFROM Society, a new social network site built on the DAEFROM blockchain, will launch till 1 year. Using blockchain technology, users will have more control over their data and privacy and the ability to earn free DAEFROM tokens by watching ads and so on.

Anonymized network blockchain

In the near future, we want to release a customized anonymized blockchain DAE Blockchain.

Custom Code

Cur codebase has been modified to meet the needs of a wide spectrum of people, with future updates planned depending on what is decided.

We are LIVE, DAE is born!

Mainnet Blockchain, after successful testing for several weeks, is launched live. The main backbone consists of several servers, which will expand with the demands of the network.

Premined Coins

We have premined about 50M coins. DAE coins will be burned and used for :

  • Development and maintenance
  • Public relations marketing
  • Redemption of DAEFROM tokens.

Roadmap

2021 4Q

Birth stage

Create platform from ethereum blockchain, build the backbone DAEFROM network with DAE mining coin and DAEFROM token, implementing core engine, create pools

2022 2Q

Testing phase

Unit Testing, Integration Testing, System Testing, and Acceptance Testing.

2022 4Q

Start of public mining and public business on markets

Create mining pools network. Start of public bussiness and listing on one of top crypto market, start of pre sale DAEFROM token.

2023 2Q

DAEFROM Token

Start rewarding mined coins with the DAEFROM token, the reward will be retroactive. Create staking pool for token, establishing on one of top crypto market.
